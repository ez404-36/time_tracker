## Описание

TimeTracker — ваш личный помощник в работе без лишнего шума.
Отслеживайте, как вы проводите время, не теряя контроля над рутиной.
С приложением можно следить за бездействием, активными окнами и работать по Pomodoro-таймеру — всё это настраивается под ваш стиль.
Добавьте задачи в TODO-лист и оставайтесь в фокусе.

## Установка

#### Готовая portable-версия
Просто скачайте и запустите готовый исполняемый файл в соответствии с вашей ОС.

#### Ручная установка
Требуется: наличие пакетного менеджера `uv` для `python`.
Установка и запуск:
1. Загрузите исходный код приложения с GitHub
2. Выполните следующую команду в корне проекта: `uv sync`
3. Активируйте виртуальное окружение `uv`.
4. Запуск: `python main.py`

## Дополнительная информация

### Linux

#### Различные оконные системы

Чтобы узнать используемую оконную систему, введите
```bash
echo $XDG_SESSION_TYPE
```

##### X11
Для оконной системы x11 требуется установить доп пакет:
```bash
sudo apt install xprintidle
```
С помощью него система корректно сможет отображать время бездействия.

##### Wayland
Для оконной системы Wayland требуется активировать `unsafe_mode` для работы трекера активных окон:
1. `Alt + F2`
2. `lg` -> Enter
3. `global.context.unsafe_mode = true` -> Enter

### Windows
Для корректной работы звуковых уведомлений убедитесь, что у вас установлена программа `ffmpeg`.
Установка:
```shell
winget install ffmpeg
```
